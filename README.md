## Features
- Real-time technical analysis (RSI, MACD, EMA, Bollinger Bands)
- Clear trading signals with support/resistance levels
- Strategy backtesting + equity curve
- Fully automated daily alerts via Discord (07:30 ICT)
- Built with production-grade tools

## Tech Stack
- Python, pandas, yfinance, ta
- Supabase (PostgreSQL)
- Streamlit + Plotly
- GitHub Actions + Discord Webhooks
